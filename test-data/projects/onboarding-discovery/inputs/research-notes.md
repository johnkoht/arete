# Research Notes — Onboarding Discovery

## Interview Themes (5 interviews, Jan 2026)

### Step 3 Drop-off
- 3 of 5 interviewees stalled at step 3
- Common feedback: "Not sure what to do next"
- One user: "I thought I was done but it said I had more steps"

### Admin Setup
- All 5 mentioned admin configuration
- 4 said they looked for in-app help first
- External docs described as "hard to find" and "out of date"

### In-App Guidance
- 4 of 5 requested contextual tooltips or checklists
- Preference for step-by-step in product over linking to docs
- One power user: "Docs are fine for reference, but I need help in the moment"

## Support Ticket Analysis

- Top 3 ticket categories: onboarding step 3 (28%), admin config (22%), account setup (18%)
- Average time to resolution: 3.2 days for onboarding, 1.8 for admin
- Self-serve attempts before ticket: 65% tried help center first
