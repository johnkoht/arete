### 2026-01-31: Onboarding step 3 is primary drop-off
**Source**: User Research Synthesis
**Insight**: 3 of 5 interviewees stalled at onboarding step 3; confusion about "what's next" is common
**Implications**: Prioritize step 3 simplification in onboarding v2; add clear progress indicators
**Applied To**: onboarding-discovery scope, Q1-1 planning

### 2026-01-31: Users prefer in-app guidance over docs
**Source**: User Research Synthesis
**Insight**: 4 of 5 users requested contextual help; external docs are secondary
**Implications**: Design in-app tooltips, checklists, and step guidance; docs remain for reference
**Applied To**: onboarding-discovery recommendations
