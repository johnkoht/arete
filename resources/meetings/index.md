# Meeting Log

> **Purpose**: Chronological index of all meetings. Raw meeting notes are stored here as immutable resources.

---

## How to Use

- Add meetings to the log below when they occur
- Individual meeting files: `YYYY-MM-DD-topic.md`
- Decisions from meetings should be extracted to `memory/items/decisions.md` with a source link
- Projects can reference meetings: `[meeting](../../resources/meetings/YYYY-MM-DD-topic.md)`
