# Notes Log

> **Purpose**: Index of standalone notes and captures that aren't tied to a specific project. These are timestamped, permanent records.

---

## How to Use

- Add notes to the log below when captured
- Individual note files: `YYYY-MM-DD-topic.md`
- Use tags to categorize for easier retrieval
- For temporary working notes, use `scratchpad.md` instead
